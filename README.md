# The Casefilers

A four-filer underground public-source intelligence unit.

> Reconstruct the record. Build the case.

---

## What this repository is

The implementation of The Casefilers — a software product framed as an intelligence unit. Users open cases; the unit convenes; the room seals; the file is bound; the user reads the finding.

This is not an AI assistant. It is a unit. See `CLAUDE.md` and `docs/01_brand_bible.md` before contributing.

## Non-negotiable rules

- The unit is **never** called AI, agents, or models in user-facing copy
- The voice register (institutional / Oracle / filer working) is locked — see `docs/02_oracle_voice_guide.md`
- The dossier palette is canonical — see `docs/05_engineering_handoff.md`
- The filer badge always holds the filer's personal color, even during operations mode
- Two materials never blend (warm dark wood / cream paper / operations green)
